⚠️ Danger Zone

Reset All Stats
Permanently deletes all chat sessions, messages, token usage records, and audit log events. User accounts, Prompt Security regions, app settings, and demo scenarios are preserved. This cannot be undone.
Reset Database
⚠ EXTREME CAUTION. This will permanently destroy all data in the database — including every user account, Prompt Security region, chat session, message, audit event, API key, demo scenario, and all application settings (encryption key, JWT secret, admin password, LLM keys, email config, and more).

The application will be reset to factory state. All settings must be reconfigured from scratch via the Settings panel.

There is no backup. This action is completely irreversible.
